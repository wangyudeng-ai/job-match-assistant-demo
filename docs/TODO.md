# 求职匹配插件待办清单

## 当前结论

V0.1 直接做 Boss 直聘职位详情自动捕捉原型。

核心闭环：

1. 用户上传 PDF 或纯文本简历。
2. 插件提取简历文本。
3. 用户输入目标岗位。
4. 用户打开 Boss 直聘职位页面。
5. 插件自动捕捉当前职位信息。
6. 插件基于简历、目标岗位和职位信息生成匹配分析。
7. 捕捉失败时，用户可以手动粘贴 JD 兜底。
