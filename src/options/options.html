本地后端

简历结构化和职位匹配分析都通过 http://localhost:3000 完成。API Key、模型名和 Base URL 只在项目根目录的 .env 中配置，不再保存在插件里。

数据与隐私

本地保存

  • 目标岗位
  • 简历原文和结构化简历
  • 最近 20 条职位分析历史

发送给本地后端

  • 简历结构化：简历原文
  • 匹配分析：简历、目标岗位和当前职位
  • 模型配置只从后端 .env 读取

当前阶段

  • 已支持目标岗位本地保存。
  • 已支持 Boss 页面自动捕捉职位信息。
  • 已支持纯文本和文本型 PDF 简历文本提取。
  • 已支持本地后端调用模型生成可编辑的结构化简历。
  • 已支持本地后端调用模型生成匹配分析，并保存最近分析记录。
  • 已支持删除简历数据、清空分析历史和查看数据使用说明。
